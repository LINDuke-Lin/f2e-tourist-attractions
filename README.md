### cz 說明

feat: 功能相關（feature）

fix: 修 bug

docs: 文件（README.md 之類）

style: 程式樣式（不影響程式運行的變動，例如空格等，注意不是 css）

refactor: 重構（不是新增功能，也不是修改 bug 的程式變動）

test: 測試

chore: 構建過程或輔助工具的變動，例如安裝套件，或是設定 vite.config、.env 等設定檔
